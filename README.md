# Chess!

> A multiplayer chess game with a live chat feature!

### Installing

```
npm install
```

## Built With

* [React](https://reactjs.org/)
* [Socket.IO](https://socket.io/)
* [Node.js](https://nodejs.org/en/)
* [Express](https://expressjs.com/)
